## 📌 소개
이 프로젝트는 **BRAF 변이를 가진 A375 흑색종 세포주 기반의 세포 생존율 분석 및 데이터 시각화 도구**입니다. 
흡광도(Absorbance) 측정 결과나 실험 데이터를 기반으로 세포 생존율을 자동으로 계산하고, 정량적인 통계 분석 및 그래프화 프로세스를 자동화하여 연구 효율성을 높이기 위해 개발되었습니다.

* **타겟 사용자:** 세포 생존율(MTT, CCK-8 등) 데이터의 빠른 시각화와 통계 처리가 필요한 바이오 연구원 및 학생.

## ✨ 주요 기능
- **💡 세포 생존율 자동 계산:** 흡광도 raw 데이터를 입력받아 대조군(Control) 대비 세포 생존율(%)을 자동으로 산출합니다.
- **📊 통계 기반 시각화:** 분석된 데이터를 바탕으로 IC50 값 추정, Bar 그래프 및 Dose-Response Curve(농도-반응 곡선)를 자동으로 생성합니다.
- **⚡ 데이터 필터링 및 전처리:** 이상치(Outlier)를 감지하고 실험 오차를 최소화할 수 있는 데이터 정제 기능을 제공합니다.
